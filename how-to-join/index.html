How to Join BalkAnarchy

BalkAnarchy is a modded Minecraft server that requires a custom modpack to play. Follow the steps below to get started.

Download the Modpack

Our modpack is hosted on Sewt's Modpack Repository. You can download it via the following platforms:

  • CurseForge
  • PrisMC
  • Modrinth App

Manual Installation

If you prefer, you can download the mods directly and set up the modpack yourself:

Setup Instructions

  1. Download and install one of the recommended platforms (CurseForge, PrisMC, or Modrinth).
  2. Import the modpack using the platform's import feature.
  3. Launch Minecraft and ensure the correct version is selected.
  4. Join the BalkAnarchy server and enjoy!
